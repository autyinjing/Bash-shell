
bash学习笔记：

注：以下的命令只列出了部分较常用的选项，关于命令的更多内容请参考man手册；

一、认识shell

1、什么是shell？
   shell是一个 用户能够操作应用程序 的接口；

2、为什么要学习shell？
   ①通用性：几乎各个Linux发行版使用的bash都是一样的，学会一个就可以“一通百通”；
   ②远程管理：Linux的管理经常需要远程联机，而联机时命令行界面的传输速度比较快；
   ③Linux的任督二脉：使用shell可以更接近Linux，良好的shell程序编写可以更好的管理主机；

3、shell种类？
   bash是Linux默认的shell；
   可以通过查看 /etc/shells 文件来了解可用的shell，常用的有：sh、bash、ksh、tcsh、csh、zsh

4、bash的功能？
   ①命令记忆：～/.bash_history中记录着前一次登录以前所执行过的命令；
   ②命令与文件补全：按tab键可以自动补全命令或者文件名；
   ③命令别名设置：alias可以设置命令别名；
   ④作业控制、前台、后台控制：如可以使用 & 让任务在后台执行；
   ⑤程序脚本：shell scripts可以通过交互的方式整合多个命令在一个脚本文件中，方便管理Linux；
   ⑥通配符：如 "*" "?" 等通配符可以大大加快用户操作； 

5、bash的内置命令：type
   type [-tpa] name     #功能：查看name指定的命令是外部命令、bash内置命令还是命令别名；

二、shell变量

1、什么是变量？
   用一个名字来代替复杂或者容易改变的数据；

2、变量的显示与设置？
   name=val     #将name指定的变量的内容设置为val
   echo $name  
   echo ${name} #显示name对应变量的内容
   unset name   #取消name对应变量的设置
   应用之一：将经常要切换的较深的目录的完整路径设置为一个变量name，然后可以通过cd $name切换到name指定的目录；

   env          #列出目前shell环境下所有的环境变量与其内容
   set          #查看环境变量与自定义变量
   export name  #将name指定的变量转换成环境变量

3、变量的有效范围？
   自定义变量=局部变量：有效范围为当前进程
   环境变量=全局变量：有效范围为当前进程和其子进程

4、变量键盘读取、数组与声明？
   read [-pt] name      #从标准输入获取变量的内容，[-p "tip"]设置显示tip指定的提示符，[-t sec]设置定时器
   declare [-aixr] name #声明name指定的变量类型，a、i、x、r分别对应：数组、整数、环境变量、只读（不能更改和重设）

5、与文件系统及程序的限制关系：ulimit
   ulimit -a    #列出目前身份的所有限制数据数值
